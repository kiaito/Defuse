## Defuse the Bomb guessing game
### Mobile Application
A guessing game to cut the correct color wire to defuse the bomb.
- The game has 3 Levels 
- 1 Less wire with each level

### Visual Studio Xamarin Forms
- IOS
- C#
- XAML
